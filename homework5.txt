2. What are the difference between value type and reference type variables?
value type is the data stored directly by its value, while reference type hold memory address for the value
value type is not collected by garbage collector while reference types are collected by garbage collector
value type store in stack while reference type store in heap

statement?
break means break out of the entire loop; return means break and return value to the method; continue means jump to the head of the next loop
